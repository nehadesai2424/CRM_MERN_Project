##  Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/nehadesai2424/CRM_MERN_Project.git
cd CRM_MERN_Project
```

### 2. Backend Setup
```bash
cd backend
npm install
npm start
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
npm start
```
